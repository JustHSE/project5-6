# Library Management System (C++)

Консольное приложение для управления личной библиотекой на C++.

## Меню

```
=== БИБЛИОТЕКА ===
1. Просмотреть все книги
2. Просмотреть всех пользователей
3. Добавить новую книгу
4. Зарегистрировать пользователя
5. Выдать книгу пользователю
6. Принять книгу от пользователя
7. Поиск книги по ISBN
8. Просмотреть профиль пользователя
9. Сохранить данные в файл
10. Выход
Ваш выбор:
```

## Функции

Просмотреть все книги - вывод каталога всех книг с информацией о названии, авторе, году издания, ISBN и доступности

Просмотреть всех пользователей - отображение списка зарегистрированных пользователей и их профилей

Добавить новую книгу - ввод названия, автора, года издания и ISBN с автоматической валидацией года

Зарегистрировать пользователя - добавление нового пользователя с фамилией, ID и лимитом книг

Выдать книгу пользователю - передача книги пользователю с проверкой лимита и доступности

Принять книгу от пользователя - возврат книги в каталог библиотеки

Поиск книги по ISBN - быстрый поиск конкретной книги

Просмотреть профиль пользователя - отображение информации о пользователе и его взятых книгах

Сохранить данные в файл - сохранение состояния библиотеки для последующей загрузки

Выход - завершение программы

## Хранение данных

Данные сохраняются в файле data/library_data.txt в формате:

```
BOOK
Title: Название книги
Author: Автор
Year: 1869
ISBN: 978-5-17-70490-8
Available: yes
BorrowedBy: 

---USERS---

USER
Name: Фамилия
UserID: USR_001
BorrowedBooks: 978-5-17-70490-8
MaxBooks: 3
```

## Технические особенности

Максимальное количество книг: неограниченное

Максимальное количество пользователей: неограниченное

Используются стандартные библиотеки: iostream, fstream, string, vector, stdexcept, iomanip

Объектно-ориентированное программирование с использованием трех основных классов

Автоматическая загрузка данных при запуске приложения

Валидация года издания (1450 - текущий год)

Проверка уникальности ISBN и фамилии пользователя

Обработка исключений при ошибках валидации

Система указателей для поиска объектов

Сохранение и загрузка состояния из текстового файла
